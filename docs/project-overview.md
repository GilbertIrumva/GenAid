# Generation Aid Website Redesign

## Project Vision

Redesign the Generation Aid website into a modern, accessible, responsive, and scalable digital platform that communicates impact, attracts partners, supports fundraising efforts, showcases programs, and highlights refugee-led innovation in Kakuma.

## Objectives

- Improve user experience and navigation
- Increase credibility and trust
- Showcase impact through data and stories
- Improve mobile responsiveness
- Create a scalable content management workflow
- Build an admin dashboard for content updates

## Target Users

### Primary Users

- Donors
- Partners
- NGOs
- Employers
- Volunteers

### Secondary Users

- Refugees
- Program beneficiaries
- Community members
- Researchers
